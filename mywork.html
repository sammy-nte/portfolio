React Travel Journal

React was used to create this app. It shows a list of places I've visited, and there's a local database called 'data.js' that contains vital information about the locations featured in the app. To properly run the app, data from 'data.js' is fetched and the necessary actions are taken. Credit to Scrimba for this project. It was a solo project under their Learn React course

This project makes extensive use of CSS animations and JavaScript. By making this project, I put my CSS animation skills to the test, and while it isn't visually appealing, I learned a lot from it. Users can change the colors of specific parts of the project to the given colors or a color of their choice with the help of some JavaScript implementations behind the scenes.

This is a simple UI i designed for the students at Young Geeks Technology who where learning mobile app development.
